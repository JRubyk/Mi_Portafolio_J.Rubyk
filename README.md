# 🍂 Portafolio Personal — Jeimy Cáceres

Portafolio profesional de **Jeimy Cáceres**, desarrolladora web con base sólida en frontend y creciente expertise en backend Full Stack Python.

---

## ¿Qué es este proyecto?

Sitio web estático de una sola página (OnePage) que presenta mi perfil profesional, habilidades técnicas, proyectos desarrollados e información de contacto.

Refleja mi identidad como desarrolladora: diseño funcional, experiencia de usuario cuidada y código limpio con propósito.

---

## ¿Con qué está hecho?

- **HTML5** — estructura semántica
- **CSS3** — estilos personalizados, animaciones y diseño responsive
- **JavaScript** — interactividad, efectos de scroll y animaciones
- **Devicons** — íconos de tecnologías vía CDN
- **Google Fonts** — tipografías Cormorant Garamond e IBM Plex Mono

Sin frameworks. Sin dependencias de instalación. HTML puro listo para GitHub Pages.

---

## ¿Cómo verlo?

### Opción 1 — Abrir directamente
Descarga o clona el repositorio y abre `index.html` en tu navegador.

### Opción 2 — GitHub Pages
Visita el sitio publicado en:
```
https://jrubyk.github.io/portafolio
```
